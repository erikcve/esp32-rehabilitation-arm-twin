# Hardware

| Function | ESP32 Pin / Peripheral |
| --- | --- |
| MPU6050 SDA | GPIO21 |
| MPU6050 SCL | GPIO22 |
| Potentiometer ADC | GPIO34 / ADC1 Channel 6 |
| Shoulder horizontal servo | GPIO25 |
| Shoulder vertical servo | GPIO26 |
| Elbow servo | GPIO27 |

Use an external supply for the servos and connect grounds between the servo supply and ESP32.
